Radius / Rocker Marchese 表盘读数说明

冰刀弧度与 Bend 一样用 Marchese Gauge 记录表盘读数。以中心点 0 为基准，往刀头方向标为 +1 到 +10，往刀尾方向标为 -1 到 -10。通常中心附近 2-3 个点可能接近，其余测量点会逐步变化；+9、+10、-9、-10 可根据刀长和量具接触情况选填。

Offset 横向偏置测量说明

Offset 以鞋底纵向中心线或脚掌受力中心线为基准，测量刀刃中心线相对该线的横向偏移。建议在前掌安装点和后跟安装点各量一次。记录时用“内侧/外侧 + mm”，不要只写左/右，避免左右脚方向相反造成混淆。

Bend / Marchese 表盘读数说明

Marchese 表盘每小格为 0.0001 inch。实际记录通常使用教练口头简写，例如“18”表示 0.0018 inch，“22”表示 0.0022 inch。请记录表盘读数，不要写成 mm。压弯后刀片弯度不是单一数字，应按刀身头段、中段、后段分别记录。头段影响入弯、起始咬冰、转向进入感；中段影响弯中支撑、主承重稳定性；后段影响出弯、推蹬释放、尾部稳定性。
